# game-soul-mahjong-tutorial

雀魂絶藝總綱(雀魂绝艺总纲)

> Laya 引擎搭建项目

## 版本

> node ^v14.17.4
> npm ^6.14.6
> cnpm ^8.1.0
> laya 1.7.19

## 项目编译和启动

1. vscode 打开项目
2. 安装 layaair-cmd 环境

```cmd
cnpm i gulp -g
cnpm i layaair-cmd -g
```

3. 命令行运行`layaair-cmd.cmd comile

## 项目目录

```text
---
  --.laya // laya引擎脚本配置
  --.vscode // vscode引擎开发脚本
  --bin //源文件
    -- data // 章节相关json文件
    -- libs // laya 库文件
    -- res // 资源素材文件
  --src
```
